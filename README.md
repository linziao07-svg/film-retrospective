# 私人电影回溯小程序 V1

这是一个本地静态版原型，直接打开 `index.html` 即可使用。部署到 Netlify 后，可以在手机浏览器里添加到主屏幕。

## 已实现

- 底部三栏：滑动区、搜索区、资料区。
- 默认打开进入滑动区。
- 滑动区：真正拖拽电影海报，左滑没看过，右滑看过。
- 中场休息：每 15 张卡片后，左滑回收加入待看，右滑选择“我喜欢”。
- 搜索区：同时检索本地资料和外部电影资料。
- AI 推荐：放在搜索区，支持小众优质、经典老片、冷门国家、相似气质。
- 资料区：默认展示所有已看电影，也可查看我喜欢和待看。
- 电影详情：片名、国家/地区、导演、上映年份、海报、模糊观看时间、标签、备注、我喜欢介绍。
- TMDb：通过 Netlify Functions 中转 `/api/discover` 和 `/api/search`，不在前端暴露 key。
- PWA：包含 `manifest.webmanifest`、`service-worker.js` 和应用图标。

## 数据

数据保存在浏览器本地存储中，刷新页面不会丢失。当前版本不做账号和云同步。

## 配置 TMDb

部署到 Netlify 后，在站点设置里添加环境变量：

- `TMDB_API_KEY`：TMDb API key，或
- `TMDB_ACCESS_TOKEN`：TMDb v4 access token

没有配置时，小程序会使用内置候选兜底，但外部搜索和持续补货能力会变弱。

## 放到手机上

最直接的方式是把整个文件夹部署成静态网站，例如 Vercel、Netlify 或 GitHub Pages。

部署后：

1. 用手机 Safari 或 Chrome 打开部署后的网址。
2. iPhone 点分享按钮，选择“添加到主屏幕”。
3. 安卓 Chrome 点菜单，选择“安装应用”或“添加到主屏幕”。

注意：当前数据保存在各自设备的浏览器本地。电脑和手机暂时不会自动同步，后续需要接云端数据库。
